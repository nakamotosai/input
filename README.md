# AI Japanese Input Method (AI 日语输入法)

AI 驱动的下一代语音输入与中日翻译工具，结合极简美学与硬核性能。

## ✨ 核心特性 (Key Features)

### 1. 双重核心模式

- **🎙️ 中文识别模式 (Chinese ASR)**:
  - 快速、高精度的中文语音转文字。
  - **老虎机动画**: 识别过程中呈现丝滑的文字滚动与对其效果（居中对齐）。
  - **自动过滤**: 智能识别有效人声，自动忽略背景噪音和空白录音，杜绝乱码。

- **🈷️ 中日双显模式 (Translation Mode)**:
  - 专为日语交流设计。
  - **实时双语**: 上半部分显示日语翻译结果，下半部分保留中文原文。
  - **UI升级**: 左对齐布局，精致的“日>”、“中>”角标。
  - **彩虹分割线**: 动态滚动的彩虹光效分割线，视觉体验极佳。

### 2. 🌈 彩虹声能条 (Rainbow Spectral Bar)

- 当主界面隐藏时，启用全新的**悬浮录音指示器**。
- **极简设计**: 平时仅为一条 4px 高的隐形细线，不遮挡屏幕。
- **动态响应**: 说话时随音量横向伸展，并呈现**流动的七彩虹光**。
- **无感体验**: 释放快捷键即刻上屏/翻译，过程行云流水。

### 3. 本地化与性能

- **SenseVoice 引擎**: 本地运行的高精度 ASR 模型，无需联网，隐私安全。
- **性能优化**: 针对大段文本和高频输入进行了渲染层优化，拒绝卡顿。
- **智能标点**: 自动处理标点符号，智能断句。

## ⌨️ 快捷键 (Hotkeys)

- **`Win + Ctrl` (默认)**: 按住说话 (语音输入/翻译)。
- **`Win + Alt`**: 显/隐主界面。
- **双显模式切换**: 在托盘图标右键菜单中选择“中日双显模式”。

## 🛠️ 安装与使用

1. **运行**: 直接启动 `main.py` 或打包后的 `exe`。
2. **模型**: 首次运行将自动检查 `SenseVoiceSmall` 模型。
3. **设置**: 可通过托盘菜单调整透明度、字体（支持思源黑体/宋体）和应用模式。
